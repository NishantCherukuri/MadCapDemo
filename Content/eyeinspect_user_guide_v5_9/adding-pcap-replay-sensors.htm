Adding PCAP Replay Sensors

When working on a Bundled Configuration, the option to add a PCAP Replay Sensor is available. To add a PCAP Replay Sensor:

  1. Navigate to the Sensors Overview page and select "New sensor > Add PCAP replay sensor"
  2. Enter the PCAP Replay Sensor's details in the modal form
  3. Click "FINISH"
A PCAP Replay Sensor will be created by the eyeInspect Command Center.
Note: Once a PCAP Replay Sensor is removed, its port number may be re-used. If multiple PCAP Replay Sensors are used and the same IP address is observed by multiple Sensors, by default eyeInspect will consider that IP address as a single asset. Set up IP reuse domains to prevent eyeInspect from merging assets based on those IP addresses. The same applies to when a Passive Sensor is listening to a network with identical IP addresses.
