Reports Settings

The following reports setting is available:

Manage Reports

The Manage Reports page can be used to check scheduled reports and associated details such as:

  • Report name
  • Report type
  • Information
  • Scheduled on
  • Recipient(s)
  • Status
  • Scheduled by
  • Email server

Use the Actions dropdown to pause, resume or delete reports.

Use the SCHEDULE A REPORT button to schedule a new report.
