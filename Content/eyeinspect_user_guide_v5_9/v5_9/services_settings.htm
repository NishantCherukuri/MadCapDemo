Services Settings

Services settings are available only in appliances running bundled configurations. The following services setting is available:

PCAP replay

The PCAP replay setting allows you to import a PCAP capture from your local machine and replay it. In addition, the Replay log table at the bottom shows the details of the PCAP replay.

The following options are available in the secondary navigation bar:

Back

Go back to the previous page.

Reload

Reload the PCAP replay data on the page.

Clear data

Clear the PCAP replay data on the page.

Add PCAP replay sensor

Add a new PCAP replay sensor. For more information, go to Adding PCAP Replay Sensors.
