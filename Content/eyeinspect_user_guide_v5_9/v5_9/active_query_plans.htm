Active Query Plans

The Active Query Plan page shows a list of active query plans and associated details in a tabular format. Click a row to open the Create an Active Query Plan window to edit the query plan.

To perform actions on one or more active queries, select the respective checkbox(es) in the first column to activate the action icons above the table -

The actions are, in order:

  • Copy the query plan details
  • Start the active query plan
  • Stop the active query plan
  • Delete the active query plan
  • Reload the active query plans table

Create an Active Query Plan

Click Create an Active Query Plan from the secondary nav bar to open the Create an Active Query Plan dialog and follow the on-screen prompts.

Edit an Active Query Plan

From the Active Query Plans table, click any column of an active query plan (except the checkbox in the first column) to open the Edit Active Query Plan dialog. Here, you can edit the Configuration, Target and Options of the active query plan.
