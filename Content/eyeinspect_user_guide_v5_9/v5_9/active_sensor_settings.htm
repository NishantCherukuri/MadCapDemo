Active Sensor Settings

The Active Sensor page displays the following information:

Sensor attributes

A summary of the sensor attributes, including the sensor name, state, address, port, target networks, IP reuse domains and monitored networks is displayed.

Scan tasks

This table shows all the active scan tasks with various details.

The following options are available in the secondary navigation bar on this page:

Back

Navigate to the previous page.

Settings

Opens the Edit Active Sensor page. Here you can edit the sensor name, IP Reuse Domains and Targetable IP Ranges.

Diagnostics

Displays the various sensor diagnostics details such as health status of the CPU, memory and disks.

In addition, it is possible to distribute new active fingerprinting bundles to all connected active sensors:

Distribute Active Fingerprinting Bundles to Connected Active Sensors

Active fingerprinting bundles distribution happens via the CLI. To do this:

  1. Download the latest active fingerprinting bundle (named eyeInspect-active-sensor-fingerprinting.zip) to the Command Center.
    Note: Change the name of this bundle as there is already a default bundle present at /home/silentdefense with the same name.
  2. Run the following command:
    sudo nse-import <file-name.zip>

Note: The active fingerprinting bundle can only be distributed to connected sensors. If a sensor is connected or added later, run the command again to distribute the bundle to the newly connected or added sensors.
