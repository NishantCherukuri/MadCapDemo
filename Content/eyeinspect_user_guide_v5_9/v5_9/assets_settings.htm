Assets Settings

The following assets settings are available:

Custom properties

Besides the standard asset properties which eyeInspect supports, users can also create and assign custom ones. Custom properties' names must be unique and their values must be textual or numeric. The maximum amount of custom properties supported at the same time is limited to 20. They can be created, edited and deleted from the Custom Properties page, which can be reached from the Secondary Navbar on the Assets Page. Custom Properties can be displayed as columns in the Assets Table, as long as the user enables them from the Columns drop-down menu above the table.

Similarly, the assets on the page can be filtered by the custom properties, which should appear in the "More Filters" drop-down under the section "Custom Properties". Custom Properties can also be assigned to one or multiple assets via the "Edit Multiple Assets Properties" action from the Actions drop-down.

Assets forwarding

eyeInspect can be configured to forward the assets information to an external server of the user's choice. Use the Assets Forwarding page to enable asset forwarding, and fill in the respective configuration and message details. Use the TEST CONNECTIVITY button to test the flow.

Purdue model

Use the checkbox to enable or disable Purdue model for eyeInspect assets.
