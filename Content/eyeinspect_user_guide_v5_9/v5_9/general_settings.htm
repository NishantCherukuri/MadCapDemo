General Settings

The following general settings are available:

  • eyeInspect Playbooks: Links to the available eyeInspect Playbooks
  • Date and Time: The date and time of the Command Center can be manually modified or synchronized with NTP servers. To edit the date and time of a Sensor, visit the corresponding Manage Sensor page.

  • Edit timeout: The timeout period (in seconds) for all charts and tables on the Home and Assets pages.
  • Login disclaimer: A custom Login page disclaimer can be setup to be displayed to all users accessing the eyeInspect Command Center web interface. Please note that the disclaimer text is read as plain text. Any HTML formatting will be treated as text.
  • Change language: Allows the user to choose between English (US), English (UK) and Japanese. The login page is displayed in the default language, English (US).
  • CSV delimiter: Allows the user to select the delimiter for CSVs. A CSV delimiter is a character that separates values within a CSV file. The selected delimiter will apply to all exports and imports from and to the Command Center. The available options are:
    • Comma (,)
    • Semicolon (;)
