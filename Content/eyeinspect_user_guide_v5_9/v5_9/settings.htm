Settings

Click SETTINGS from the primary navigation bar to access the following two sub-menus:

System Settings

The following system settings are available:

  • General Settings
  • Maintenance Settings
  • Services Settings
  • Events Settings
  • Active Scans Settings
  • Assets Settings
  • Users Settings
  • Risks and Threats Settings
  • Reports Settings

Import

The Import sub-menu allows you to do the following:

  • Import Assets CSV: Upload a CSV file with asset details to populate the monitored hosts.
  • Import Asset Tab: Upload a JSON file with asset filters to obtain a desired predefined Assets Tab.
  • Import PCAP Replay: Upload a PCAP, PCAPNG or ZIP file with traffic capture(s) to replay it with a PCAP Replay Sensor. Note that this option is only available in Bundled Configurations.
  • Import Alert Tab: Upload a JSON file with alert filters to obtain a desired predefined Alert Tab.
