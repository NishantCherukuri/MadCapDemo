Command Center Diagnostics

The Command Center Diagnostics page details about the Command Center's CPU, memory and disk usage and system components. It also gives easy access to several logs that are useful for troubleshooting. In addition, all diagnostics of the Command Center and those of selected Sensors can be exported easily using the "Export" button on the sub-menu of the Command Center diagnostics page.

A zip file containing all logs and diagnostics entries of the Command Center and all logs, diagnostics entries, a list of profiles, modules and ITL check configurations of the selected Sensors will be generated and made available for download from the "Exported diagnostics" table at the bottom of the Diagnostics page. The status of a diagnostics category can be "normal", "warning" or "critical".

The Command Center Diagnostics page can be accessed by clicking on the Command Center icon in the top right corner of the main menu. If a category requires the user's attention the Command Center icon will be highlighted with an exclamation mark.
