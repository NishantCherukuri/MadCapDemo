Host Editing (Monitored Hosts)

The monitored hosts page lists all the available monitored hosts in a tabular form.

The following features are available in the secondary nav bar:

Reload

Use this option to reload the monitored hosts table.

Add

Use the Add option to define a new host. Please note the instructions for the "Address" and "VLAN IDs" fields by reading the help available by hovering over the question mark icon at the right side of the text field. Note that a sensor can be associated with a monitored network in the sensor's settings. In addition, monitored networks for hosts which do not fall into any current network can be generated automatically by using the "Create missing networks" option. For more information, go to the Map section of the Asset Inventory page and see the Actions paragraph.

Filter

Use the Filter option to define advanced filters.

Export

With the Export option, you can:

  • Use the Export > Report sub-menu to define what should be present in your report.
  • Use the Export > CSV Export sub-menu to export all monitored hosts as a CSV file. Users can export the existing monitored eyeInspect hosts, edit them offline, and import them back into the Command Center. Note that any modifications to the "Monitoring sensors" column in the exported CSV file will be ignored when the CSV file is imported back into eyeInspect.

Edit and Delete

Select one or more monitored hosts from the table by selecting the checkbox in the first column to enable the Edit and Delete options in the secondary nav bar.
