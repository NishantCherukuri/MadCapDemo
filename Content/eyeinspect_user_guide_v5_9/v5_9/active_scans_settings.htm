Active Scans Settings

 

The following active scans setting is available:

Scan forwarding

The Scan forwarding page ("Settings" > "System Settings" > "Scan forwarding") allows users to send e-mail notifications when Active Sensor scans are started, stopped and completed. The configuration steps include filling in the basic information of the e-mail server and the connectivity info. This includes:

  • the e-mail server address and port

  • the from address and sender name

  • the recipient list

  • security settings

  • authentication information

After this information has been filled in, you can click the "Test connectivity" button to verify the connection with the server can be established successfully.

From the email received, click "View scan details in Command Center" to access the Command Center Scans overview page filtered by this specific scan, to quickly track its status or analyze the results.
