Threat Scenarios (Classic Map)

The Threat Scenarios (Classic Map) option opens the Network map page. The network map is a visual representation of the network as monitored by Sensors. This interactive graph shows all hosts and all communication between these hosts. Network maps are displayed for all asset tabs.

The following features are available in the secondary nav bar:

Reload

Reloads the network map.

Options

Opens the Map options, which allow you to display or hide host names, cluster or uncluster similar nodes, group hosts and select a layout.

Export

Allows you to export the following:

  • Asset Reports
  • Hosts CSV
  • Map PNG
  • Tabs

Tab

Allows you to perform the following tab actions:

  • Add tabs
  • Rename tabs
  • Delete tabs
  • Reorder tabs
  • Share tabs
  • Edit shared tabs

Filter

Allows you to filter the network map using various filters.

Highlight

Allows you to highlight hosts and links.

Threats

Allows you to:

  • Show alerts with basic and advanced filters
  • Show inactive hosts
  • Show cross-network or cross-level flows
  • Show network changes
  • Show hosts with known vulnerabilities
