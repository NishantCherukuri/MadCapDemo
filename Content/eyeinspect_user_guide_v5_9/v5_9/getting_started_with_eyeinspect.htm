Getting Started with the eyeInspect Command Center

This section gives an introduction to the eyeInspect Command Center, which is the user interface of eyeInspect. The Command Center is used for sensor management and control and to inspect events and manage third-party system integrations.

When a new monitoring profile is setup or an existing configuration is edited, the Command Center sends the profile/configuration information to the relevant sensor. The Command Center can manage multiple sensors. The sensors send events and network analytics information to the Command Center, where the information is stored in a database for inspection. The communication between the Command Center and the sensors is encrypted. If the connection is temporarily interrupted, the sensor buffers the events until the Command Center can be reached again.

The events and sensor health information can also be forwarded to a third-party system, such as a SIEM or syslog server. The Command Center can also integrate with third-party systems, such as LDAP for user authentication.

Log In

Open a browser and enter the URL of the eyeInspect Command Center: https://ADDR where ADDR is either the IP address or the domain name of the eyeInspect Command Center appliance. The default username and password are both set to admin. After the first login, a password change is required. If a wrong username/password combination is entered three times, the account is locked for 10 minutes. These values, as well as the password requirements, can be configured in the eyeInspect Command Center settings.

Navigation Guide

The Command Center has two navigation bars: Primary (dark blue) and Secondary (light blue). Both are split into three sections: left, middle, and right, and contain the following details:

Primary navigation bar
  • Left - Logo
  • Middle - Command Center Menu Items and Sub-menu items
  • Right - Command Center Diagnostics, Sensors Overview, Alerts, Export Report, Users, and Logout
Secondary navigation bar
  • Left - Page Name
  • Middle - Page Actions
  • Right - Current Time, System Time Zone, and Help Section

Command Center Menus

The primary navigation bar contains the following menu items:

  • Home
  • Assets
  • Events
  • Analytics
  • Sensors
  • Settings

The icons are, in order:

  1. Command Center Diagnostics: The Command Center Diagnostics reports details about the Command Center's CPU, memory and disk usage and system components. To know more, go to the Command Center Diagnostics page.
  2. Sensors Overview: Takes you to the Sensors Overview page.
  3. Alerts: Takes you to the Alerts page.
  4. Export Report: Allows you to export reports. To know more, go to the Export Reports page.
  5. Username: Allows you to change the password of your account on the Command Center. The username is displayed next to the profile icon - in the screenshot above, ADMIN is the username of the account currently logged in.
  6. Logout: Logs you out of the Command Center.
