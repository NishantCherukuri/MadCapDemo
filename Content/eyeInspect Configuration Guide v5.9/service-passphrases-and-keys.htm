Service Passphrases and Keys

 

This option allows users to generate new random passphrases for the following Command Center, Active Sensor and Passive Sensor services.

Command Center services:

  • Postgres user password
  • Druid user password
  • Command Center keystore passphrase
  • Private key passphrase (in keystore)

Active/Passive Sensor services:

  • Private key passphrase
Note: Do NOT use the Service Passphrases and Keys option on Command Centers that have High Availability enabled.
