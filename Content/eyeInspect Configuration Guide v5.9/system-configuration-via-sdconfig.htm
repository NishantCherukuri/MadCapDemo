System Configuration via SDConfig

 

This section describes the SDConfig tool used to configure all eyeInspect systems.

The sdconfig tool is a configuration utility used for configuring the eyeInspect system. The sdconfig tool is launched from the command line (by running sudo sdconfig) and can be accessed either by logging in to a system via SSH, through a physical terminal in case of deployment on a physical server, or through a hypervisor client.
