Management Network Interface

Management network interface allows you to configure the management interface, both manually, in case of using a static IP, and dynamically, in case of assignment via DHCP.

When choosing "Static", you are prompted to enter the configuration. Fill in the following fields:
  1. IP address
  2. Subnet mask
  3. Default gateway
  4. DNS server(s)
  5. Search domain(s) (if any)
