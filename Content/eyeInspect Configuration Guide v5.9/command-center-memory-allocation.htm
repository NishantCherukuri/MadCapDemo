Command Center Memory Allocation

The memory allocation tool will calculate the optimal distribution of the available memory to the Command Center services, based on the total amount of available RAM, CPU cores and hard disk on the Command Center.

 
  1. The tool will temporarily stop the Command Center application and services to better calculate the available RAM (note that this is typically lower than the total system RAM).
  2. After that, an overview screen will be presented, comparing the current and the optimal memory allocation and system parameter values; select Exit to continue.
  3. In the next screen, do one of the following:
    1. Select Yes to apply the previously displayed optimal configuration; this is recommended if the optimal values are significantly higher (10% or more) than the current ones. Note that after selecting yes, it is necessary to reboot the system to apply the changes.
    2. Select Cancel to keep the current values. If deemed appropriate, memory allocation can then be reconfigured to the optimized values.
